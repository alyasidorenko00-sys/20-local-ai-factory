# Локальная ИИ-Фабрика (МСР Сервер)

Модульная система для управления ИИ-агентами с оркестратором, общей памятью и поддержкой Ollama.


## Возможности

 **Оркестратор** - управление жизненным циклом ИИ-агентов  
 **Общая память** - хранение контекста и истории для всех агентов  
 **Подпроцессы** - запуск агентов как независимых процессов  
 **Ollama интеграция** - работа с локальными LLM через REST API  
 **FakeLLM** - имитация LLM для тестирования без установки Ollama  
 **Логирование** - полная трассировка работы системы  
 **Модульность** - лёгкое добавление новых агентов
